API List

Most Feedback House modules supports APIs. APIs can be used individually or leveraging data from other APIs.

Platform APIs:

ModuleAPIDescription
AllUserThis API can Update or Create Users. This is the most used API and is recommended that you check this one first.
AllUser GETThis API retrieve Users data.
PlanningAreaThis API can Update or Create Areas.
PlanningArea GETThis API query and retrieve Areas.
PlanningBusiness DivisionThis API can Update or Create Business Divisions.
PlanningBusiness Division GETThis API query and retrieve Business Divisions.
